## Glassmorphism Login Page with Balloon Animation

A beautiful **glassmorphism-style login page** with floating **hot air balloon animations** in the background.  
The project demonstrates **CSS animation and glassmorphism effect**.

##  Features
-  Glassmorphism login box (transparent frosted glass look)  
-  Animated hot air balloons floating in the background  
-  Clean login form with username & password fields
